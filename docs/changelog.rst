Changelog
=========

This document provides a high-level summary of notable changes to the SmartSchema backend.

Versions are illustrative; consult your VCS history and release tags for precise details.

1.0.0 (Initial Public Release)
------------------------------

- Core authentication and account management.
- Schema management and inference.
- Subscription and billing integration.
- Survey creation, invitations, and response collection.
- Initial Sphinx documentation and GitHub Pages deployment.

1.x (Ongoing Enhancements)
--------------------------

- Added support for multiple survey submissions per invite.
- Introduced migration tracking fields for survey storage and cloud migration.
- Implemented cloud storage backends (Azure, SharePoint, S3) for survey responses.
- Enhanced survey response APIs for both database and cloud-backed storage.

Future Changes
--------------

Planned improvements include:

- More granular per-endpoint API documentation in Sphinx.
- Additional operational guidance for scaling and observability.
- Extended examples for common integration scenarios.
